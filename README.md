## The function
- 1、支持三种样式；
- 2、支持动态设置输入框数目；
- 3、支持更改边框颜色；
- 4、在为方框样式时，支持设置方框间间距；
- 5、支持明文、密文状态切换；
- 6、支持手动清除已输入的支付码；
